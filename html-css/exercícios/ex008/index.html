Texto Deletado

Podemos marcar um texto como excluído para indicar que ele pode ser lido, mas não considerado

Podemos também marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois.

Existe também o sublinhado, usando a tag <u> (não semântica)

Texto sobrescrito

Para inserir coisas do tipo x20+3 , usa-se tag <sup> nos caracteres númericos 2 e 0
